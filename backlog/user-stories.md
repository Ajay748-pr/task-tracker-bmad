# User Stories

## US-101 Create Task
**As a** user,
**I want to** create a task,
**So that** I can track what needs to be done.

**Acceptance Criteria:**
- User can input task title, description, and due date.
- Task is saved in the system and appears in the task list.
- Confirmation message is displayed.

**Story Points:** 3  
**Priority Level:** High  
**Epic Association:** Task Management

---

## US-102 View Tasks
**As a** user,
**I want to** view all my tasks,
**So that** I can have an overview of what I need to complete.

**Acceptance Criteria:**
- User sees a list of tasks with titles and statuses.
- User can sort tasks by due date and priority.

**Story Points:** 2  
**Priority Level:** High  
**Epic Association:** Task Management

## US-103 Update Task Status
